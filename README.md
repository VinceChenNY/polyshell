# 🚀 PolyShell

PolyShell 是一个面向新手用户的 AI 系统助手，通过自然语言即可控制电脑、部署工具、优化系统，并提供简单直观的 AI 工作流程入口。

PolyShell is an AI-powered system assistant for beginners. It allows users to control their computer using natural language, deploy tools, optimize system performance, and access simple AI workflows.

---

## ✨ Features / 功能

### 🧠 System Check（系统体检）

* 检测系统状态（磁盘 / CPU / 内存）
* 提供优化建议
* 小白也能理解的提示

Check system status (disk / CPU / memory) and provide simple optimization suggestions.

---

### 🧹 Clean（清理系统）

* 清理缓存文件
* 释放磁盘空间
* 简单安全

Clean cache and free disk space safely.

---

### ⚙️ Deploy（工具部署）

* 一键部署 AI / 自动化工具
* 提供说明、时间、大小信息
* API 工具带安全提示（防误扣费）

Deploy AI tools with descriptions, time, size, and API safety confirmation.

---

### 🎁 Bundle（AI 工作流入口）

* 一键打开 AI + 市场 + 新闻
* 提供操作步骤
* 适合新手体验

Launch a simple AI workflow with guided steps.

---

### 🌐 Multi-language（多语言支持）

* English
* 中文
* Español
* हिन्दी
* العربية

---

## 🚀 Quick Start（快速开始）

```bash

```

然后输入：

Then type:

```text

clean / 清理
fix / 修复
```
